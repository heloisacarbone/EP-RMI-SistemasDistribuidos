# EP-RMI-SistemasDistribuidos

##RODAR O SERVER

###TEM QUE ENTRAR NA PASTA BIN DO PROJETO E RODAR O rmiregistry
* Garantir que o bin do jdk está no path da variavel de ambiente
* Na pasta bin do projeto, digitar: start rmiregistry
* O start não é necessário. Vai só abrir uma nova janela do CMD

###COLOCAR UM ARGUMENTO QUE É O NOME DO SERVER QUE ESTÁ SUBINDO.
* De o play
* Ele pede para digitar o nome do server no console.
* De enter

##Rode o Cliente
* De o play
* Ele pede para digitar o nome do servidor para se conectar
